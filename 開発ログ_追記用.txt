⸻

【バグ修正】

▶ iPhone Safari 入力バグ（ルームコード）
- onChange の中で半角変換を実行していたため iOS の IME と競合
- 修正: onChange は生の値のみ保存、変換は参加ボタン押下時のみ実行
- autoCapitalize / autoCorrect / spellCheck 属性を追加

▶ 中学生の年齢が13歳にならない
- posToAge の区間定義が誤っており pos26 が10歳に計算されていた
- 修正: pos25以下を6〜12歳、pos26以上を13〜15歳に分割

▶ アイコン選択が保存されない
- SetupScreen の onAvatarChange が stale closure になっていた
- 修正: setPlayers(prev => ...) の関数型更新に変更

▶ 初めてのアルバイトが複数回発生
- Level3 フォールバックが firedCallbacks を無視していた
- 修正: Level3・Level4 でも firedCallbacks チェックを追加

▶ 家マスで家系イベント以外が発生
- preferredCategory の適用確率が 65% 止まりだった
- 修正: Math.random() < 0.65 の条件を削除、設定時は必ず適用

▶ ページリロードで進行状況が消える
- useState の lazy initializer が SSR 側で実行されていた
- 修正: useState(null) で初期化し、useEffect でマウント後に localStorage から読込

▶ 65歳で人生終了（→85歳に延長）
- posToAge の末尾計算を修正。ゴールが 85 歳になるよう全3ファイルを更新

▶ 肌色が確定で最も暗い色になる
- DNA_SKIN に濃すぎる色（#c07850, #d4916a）が含まれていた
- 修正: 全5色を明るめ統一（#f8d4a0〜#e8b878）に変更

⸻

【新機能】

▶ 人生の重要分岐点で必ず停止（強制停止システム）
- 9マス（高校受験・進路・就活・結婚・子供・起業・キャリア再選択・早期退職・老後）に強制停止
- getFirstMandatoryStop() 関数で overshoot を防止
- 選択完了後、次ターンから通常移動に戻る
- 一度選択したマスは再発火しない

▶ 結婚ルーレット（マス79）
- 告白マス（65）で「告白する」を選んでいれば 4/6 確率、そうでなければ 2/6 確率
- 専用ルーレットモーダルを実装

▶ 告白ルーレット（マス65）
- pos25〜64 の間に起きたポジティブな恋愛イベント数 = デート回数
- デート回数に応じて成功確率が変動（0回:1/6〜4回以上:5/6）

▶ 副業システム
- YouTuber / アーティストは既に職業がある場合、主職を変えず副業として適用

▶ 隠し職業（6種）
- ⚙️ 発明家: エンジニアor理系 + 起業経験 + 知名度30以上 + pos90以降
- 💎 投資家: 資産800万以上 + 投資選択済み + pos100以降
- 🚀 宇宙飛行士: 理系大学ルート + 知名度40以上 + pos135未満
- 🎮 プロゲーマー: 知名度40以上 + pos110未満
- 👁️ 陰の支配者: 資産1500万以上 + 起業 + 知名度50以上 + pos111以降
- 🛋️ 伝説のニート: 無職/バイト + 幸福70以上 + pos60以降
- 条件未達の職業は選択肢ごと非表示（存在を隠す）
- 解放時、キャリアモーダルの最上部に金色カードで出現

▶ 職業追加（3種）
- 🎭 お笑い芸人: pos40以降。文化系部活があると有利
- 🔨 大工: pos50以降。工業・専門・即就職ルートが有利
- ⚾ 野球選手: スポーツ系部活必須 + pos40〜89（35歳前まで）

▶ アバターシステム（DotAvatar.tsx）— 同一人物成長システム

旧: 年齢ステージごとに別人のようにキャラクターが変わっていた
新: プレイヤーIDから CharacterDNA を生成し、人生を通して「同じ顔」が維持される

CharacterDNA（ゲーム開始時に1度だけ決定）
  - 髪色: 7パターン（黒・茶・金など）
  - 肌色: 5パターン（明るめ統一）
  - 目の色: 4パターン
  - 髪型: 3スタイル（ショート・サイド・センター分け）

年齢変化（変わるのは体型・服装・白髪のみ）
  - middleage（50〜64歳）: 眉・髪に22%グレー混入、目元にしわ
  - elder（65歳以上）: 眉・髪に78%グレー混入、口周りにしわ、杖追加

表情（5段階・幸福度と借金に連動）
  - very_happy (85以上): 大きな笑顔
  - happy (60〜84): 笑顔
  - neutral (40〜59): 普通
  - tired (20〜39): 疲れ顔
  - sad (20未満 or 借金-500万以上): 落ち込み顔

職業別服装
  - スーツ系: 会社員・公務員・弁護士
  - 高級スーツ: 経営者・投資家・陰の支配者（ゴールドネクタイ）
  - 白衣: 医師
  - ユニフォーム: 宇宙飛行士・野球選手
  - 作業着: 大工
  - カジュアル: アーティスト・芸人・YouTuber など

人生イベントバッジ（絵文字オーバーレイ）
  - 💍 結婚済み（右上）
  - 👶 子供あり（左下）
  - ⌚ 資産1000万以上（右上）
  - 🐾 ペットあり（中左）
  - 職業アイコン（右下）
