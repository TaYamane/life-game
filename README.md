# ドキドキ人生冒険ゲーム 🎲

スマホ専用ブラウザゲーム（スーファミ時代の人生ゲーム風）

## 起動方法

```bash
# 依存パッケージのインストール（初回のみ）
npm install

# 開発サーバー起動
npm run dev
```

ブラウザで http://localhost:3000 を開く。
スマホの場合はPCと同じWi-Fiに繋げて `http://<PCのIPアドレス>:3000` でアクセス。

## ゲームの遊び方

1. プレイヤー人数（1〜4人）を選ぶ
2. 各プレイヤーの名前・絵文字・色を設定
3. 「ゲームスタート！」を押す
4. 順番にサイコロを振ってマスを進む
5. イベントやチャンスに止まるとカードが引ける
6. 全員ゴールしたら最終資産で順位決定！

## マスの種類

| マス | 説明 |
|------|------|
| 💰 プラスマス | お金が増える |
| 💸 マイナスマス | お金が減る |
| 🎲 イベント | ランダムなイベント（良いも悪いも） |
| ⭐ チャンス | ランダムな効果（マス移動あり） |
| ⛺ 休憩所 | 何も起きない安全地帯 |
| 🎯 もう一度 | サイコロをもう一度振れる |

## 技術スタック

- Next.js 14
- TypeScript
- Tailwind CSS
- スマホ縦画面専用
